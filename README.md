# Uprooted

## Creator's Name:
  Shakela Hossain

## Summary

Uprooted is a supportive, non-clinical mental wellness app that helps users externalize, track, and understand their struggles through reflection, grounding tools, and visual progress, while creating space to gently uproot what weighs them down and plant new, meaningful goals at their own pace.

## Features
  1. Emotion Check-ins
  2. Scheduler
  3. Thought Journal
  4. Daily Short exercises
  5. Progress Chart
  6. Non-clinical Emotional Patterns & Insights
  7. User Profiles
  - Login functionality through Google
  - Update your profile information and upload a profile picture
  - Personal goals
  8. Help links
  9. Forum for discussions
  10. Supportive AI assistant


## Libraries and Frameworks
1. **Frontend**:  
  - **React**: A JavaScript library for building user interfaces. [React Documentation](https://reactjs.org/docs/getting-started.html) 
  - CSS for styling
  - **UI Testing**: https://testing-library.com/docs/react-testing-library/intro/
  - **Vitest**: https://vitest.dev/guide/ 

2. **Backend**:  
  - Node.js with Express for API development
  - **Express**: A minimal and flexible Node.js web application framework for handling HTTP requests. [Express Documentation](https://expressjs.com/)
  - **MongoDB**: A NoSQL database driver for connecting and interacting with MongoDB databases. [MongoDb Documentation](https://www.mongodb.com/docs/)
  - **Mocha**: A JavaScript test framework, designed for asynchronous testing. [Mocha Documentation](https://mochajs.org/)
  - **Chai**: An assertion library used with Mocha for testing [Chai Documentation](https://www.chaijs.com/)
  - **Sinon**: A library for test stubs, and mocks to simulate various scenarios. [Sinon Documentation](https://sinonjs.org/)

3. **Miscellaneous**:  
  - OpenAI for generating positive daily quotes
  - OpenAI for AI assistant
  - Google authentication
